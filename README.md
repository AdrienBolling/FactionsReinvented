### A project with the ambition to bring highly-manageable guilds to Hytale.

Unlike the usual well-know faction plugins, I want to allow leaders to create and manage roles, as well as their permissions, create sub-groups, etc.
This could be used to model Guilds, factions, or Kingdoms.
